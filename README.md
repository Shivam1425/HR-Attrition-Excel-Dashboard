# 📊 HR Attrition Analysis Dashboard

**Excel | Business Intelligence | Data Analytics Project**

---

## 🎯 Business Problem

Employee attrition leads to increased hiring costs, productivity loss, and knowledge drain.
This project analyzes HR data to identify **high-risk employee segments** and uncover **key factors driving attrition**.

---

## 📌 Project Overview

Developed an **interactive Excel dashboard** to transform raw HR data into **decision-ready insights** using KPI tracking and dynamic filtering.

* Dataset: ~1,400+ employees, multiple HR attributes
* Focus: Attrition drivers across **tenure, salary, department, and satisfaction**
* Output: Actionable insights for HR decision-making

---

## 🛠️ Tech Stack & Skills Demonstrated

* **Excel:** Pivot Tables, Pivot Charts, Slicers, Conditional Formatting
* **Data Analysis:** Segmentation, trend analysis, KPI design
* **Business Thinking:** Translating data into retention strategies
* **Dashboarding:** Interactive and user-driven analytics

---

## 📈 Key Business Insights

* **Early Attrition:** ~43% attrition within first 2 years → onboarding gap
* **Departmental Risk:** R&D (~56%) and Sales (~38%) dominate attrition
* **Age Factor:** Peak attrition in 25–34 age group
* **Compensation Insight:** Lower salary hikes (10–15%) → higher exits
* **Satisfaction Link:** Avg. score ~2.6 correlates with high attrition

---

## 💡 Business Impact (Simulated)

* Identifying early attrition trends can help reduce churn by **15–20%**
* Targeted compensation adjustments can improve retention in key roles
* Department-level insights enable **focused HR interventions**

---

## 🚀 Dashboard Features

* Interactive slicers (Department, Gender, Education)

* Clear visualization of attrition trends
* Drill-down capability for deeper analysis
